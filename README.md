Wouvy API

## Scripts

### Start in production
```
yarn start
```

### Start in development
```
yarn run dev
```

### Build
transpile ES6 code to JS code
```
yarn run build
```

### Clean
Clean dist directory that contain transpilled code
```
yarn run clean
```

### Test
```
yarn test
```
